# Nikken Next ランディングページ (LP) デザイン案

- **テーマ**: プロフェッショナル、クリーン、モダン、信頼性

## 2. ページ構成

### ヘッダー
- **ロゴ**: 左上に配置
- **ナビゲーション**: 機能、使い方、お問い合わせ
- **アクション**: **「ログイン」ボタン**（右上に配置）

### A. ヒーローセクション (ファーストビュー)
- **キャッチコピー**: 「日本拳法の大会運営を、もっとスマートに。」
- **サブコピー**: 「オフライン対応、デュアルモニター表示。ネット環境に左右されない、安定した試合運営をサポートします。」
- **CTAボタン**: 「デモを試す」 -> 手動モニター操作画面へ遷移する
- **ビジュアル**: **[スクリーンショット 1: モニター表示画面]** - 大きなモニターに表示されたスコアボードの高品質なイメージ。

### B. このアプリが選ばれる理由 (課題と解決策)
- **課題**: 「会場のネット環境が悪くてシステムが止まる」「紙の得点板だと観客から見えにくい」
- **解決策**: 「本アプリはオフラインベース。試合中はインターネット不要でサクサク動作。PCにモニターを繋ぐだけで、プロ仕様の得点板を観客に見せることができます。」

### C. 主な機能 (Key Features)
1.  **オフラインファースト・アーキテクチャ**
    - **説明**: インターネット接続がなくても完全に動作します。データはローカルに保存され、オンライン環境に戻った時にクラウドへ同期されます。
    - **ビジュアル**: **[スクリーンショット 2: 試合一覧画面]**（「未送信」などの同期ステータスが見えるもの）
2.  **リアルタイム・クラウド同期**
    - **説明**: オンライン中は、入力した結果が瞬時にクラウドへ保存され、他の端末にも即座に反映されます。本部席とコートサイドでの情報共有もスムーズです。
3.  **プロ仕様のモニター表示**
    - **説明**: PCにサブモニターやプロジェクターを接続するだけで、美しいスコアボードを即座に表示。操作画面とは独立した全画面表示が可能です。
    - **ビジュアル**: **[スクリーンショット 3: デュアルモニター設置イメージ]**（PC操作画面とモニター表示の対比）
4.  **直感的な操作とプレビュー**
    - **説明**: 誤操作を防ぐ洗練されたUI。キーボードショートカット対応で、画面を見ずに得点操作が可能。操作画面上でモニターの表示状態をリアルタイムにプレビューできます。
    - **ビジュアル**: **[スクリーンショット 5: モニター操作画面]**（プレビュー表示やショートカットガイドが見えるもの）
5.  **大会・チーム一元管理**
    - **説明**: 大会の作成から、チーム・選手の登録、トーナメントの組み合わせまで、ブラウザひとつで完結します。
    - **ビジュアル**: **[スクリーンショット 4: 大会設定 または チーム管理画面]**

### D. アプリ版の推奨 (Download)
- **キャッチコピー**: 「完全オフライン対応のアプリ版をご利用ください」
- **説明**: 
    - ブラウザ版でもオフライン操作は可能ですが、ページ読み込み時に通信が必要になる場合があります。
    - **ダウンロード版アプリ**なら、インターネットが全くない環境でもアプリを起動してすぐに利用開始できます。
    - Windows / Mac 対応。
- **アクション**: 「Windows版をダウンロード」「Mac版をダウンロード」(ダウンロードへのリンクを置く)

### E. 導入の流れ (How It Works)
1.  **準備 (Online)**: 大会を作成し、チーム・選手を登録して対戦カードを組みます。
2.  **実行 (Offline)**: 会場でPCを開き、モニターを接続。ネットなしで試合を進行・記録します。
3.  **同期 (Online)**: 試合終了後、ネットに接続して結果をクラウドに一括アップロードします。

## 3. 必要なスクリーンショット一覧
実装にあたり、以下のスクリーンショットをご用意ください：

1.  **モニター表示画面 (メインビジュアル用)**: スコアボードの全画面表示（MonitorIndividualMatchResult または MonitorGroupResults）。
2.  **試合一覧画面**: 同期ステータス（未送信など）が表示されているダッシュボード。
3.  **管理画面**: チーム管理または大会設定画面。
4.  **モニター操作画面**: 得点やタイマーを操作するコントローラー画面。
5.  **団体戦表示**: 団体戦特有の表示画面。

## 4. 技術仕様
- **フレームワーク**: Next.js (App Router)
- **スタイリング**: Tailwind CSS
- **コンポーネント**: `src/components` 内の再利用可能なUIコンポーネントを活用
